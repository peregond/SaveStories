# Политика конфиденциальности SaveStories

Дата вступления в силу: 10 апреля 2026 г.

## 1. Общая информация

SaveStories — это приложение для локальной выгрузки медиа по ссылкам Instagram (Stories и Reels) на устройство пользователя.

Мы уважаем приватность пользователей и стремимся к минимальной обработке данных.

## 2. Какие данные обрабатываются

Приложение может обрабатывать следующие данные:

- ссылки и имена профилей, которые пользователь вводит вручную;
- данные браузерной сессии Instagram (для авторизации), которые хранятся локально на устройстве;
- загруженные медиафайлы и технические метаданные, сохранённые в выбранной пользователем папке;
- локальные технические логи работы приложения и воркера;
- данные проверки обновлений (запрос к GitHub Releases API).

## 3. Что приложение **не** делает

- не запрашивает отдельную регистрацию аккаунта SaveStories;
- не продаёт персональные данные;
- не передаёт содержимое загруженных файлов на наши серверы;
- не ведёт рекламный трекинг.

## 4. Где и как хранятся данные

Данные хранятся локально на устройстве пользователя, в том числе:

- папка загрузок, выбранная пользователем;
- папка данных приложения (напр., `%LOCALAPPDATA%/SaveStories` или `%LOCALAPPDATA%/SaveStories.WinUI.Beta`);
- локальные файлы сессии и логи.

Хранение выполняется только для работы приложения и может быть очищено пользователем вручную.

## 5. Сетевые обращения

Приложение выполняет сетевые запросы только для:

- доступа к Instagram по действиям пользователя (авторизация/выгрузка);
- проверки и загрузки обновлений (GitHub API и ассеты релиза);
- скачивания runtime-компонентов (например, Chromium/зависимости), если пользователь запускает эту установку.

## 6. Передача третьим лицам

Мы не передаём пользовательские данные третьим лицам, кроме технически необходимых обращений к сервисам, которые пользователь использует через приложение (Instagram, GitHub, инфраструктура загрузки зависимостей).

## 7. Права пользователя и удаление данных

Пользователь может в любой момент:

- удалить загруженные файлы;
- очистить папку данных приложения;
- удалить сохранённую сессию Instagram;
- удалить приложение.

## 8. Безопасность

Мы применяем разумные технические меры для снижения рисков несанкционированного доступа к локальным данным. Однако безопасность также зависит от защиты самого устройства пользователя.

## 9. Изменения политики

Мы можем обновлять эту политику при изменении функциональности приложения или требований платформ. Актуальная версия публикуется по текущему URL.
