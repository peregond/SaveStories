## SaveMe 0.6.20

- В Windows-версии добавлен отдельный раздел `Сортировка` для переноса файлов из папки `Перенос`
- Правила вида `nick = Germany (DE)` раскладывают файлы в `Germany (DE)\nick`, а полный путь можно указать явно
- Windows-сортировка запоминает блогеров, переименовывает файлы при конфликтах и копирует дайджест по странам и блогерам
