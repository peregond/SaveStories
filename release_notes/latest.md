- Windows Stories: исправлен корневой сценарий, где worker мог отдать два JSON-ответа за один запуск после ошибки закрытия браузерной сессии.

- Windows legacy client: добавлен такой же устойчивый разбор первого JSON-ответа из шумного stdout.
