## SaveMe 0.6.35

- Исправлен ввод нескольких Instagram-профилей в Windows WinUI: список теперь корректно разбирается по строкам, пустым строкам, пробелам, запятым и `;`.
- `@username` на Windows теперь приводится к нормальной ссылке Instagram так же, как на macOS.
- Во вкладках очереди и stories кнопки действий разложены сеткой, чтобы `Очистить очередь` не пропадала за краем окна.
- Вкладка Stories на Windows получила отдельную кнопку очистки очереди.
