## SaveMe 0.6.56

- Windows Stories: таймаут batch-выгрузки теперь зависит от размера очереди, поэтому длинные списки больше не обрываются через 20 минут.
- Windows Stories и Sort: очистка пустых папок теперь использует одну общую функцию, предлагает удалить только реально пустые папки и не трогает `На перенос`.
- Windows Stories: в блоке результата добавлен live-счётчик загруженных файлов и созданных папок.
- Windows Stories: исправлена Windows-сборка после выноса очистки пустых папок в общий сервис.
