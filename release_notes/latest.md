## SaveMe 0.6.6

- Релиз включает macOS-версию и Windows WinUI 3 beta
- Для Windows WinUI 3 на GitHub публикуется установщик .exe с bundled Node runtime
- В WinUI добавлены проверки и установка обновлений прямо из приложения (check/download/apply)
- Добавлены preflight-проверка готовности, экспорт диагностики и crash logging
- Для Stories, Queue и Reels добавлена кнопка «Повторить неудачные»
- Старая Windows-версия с прежним интерфейсом по-прежнему не публикуется в релизных ассетах
