## SaveMe 0.6.41

- macOS Stories: добавлена опция `Обновлять очередь из Notion`.
- При включённой опции перед запуском Stories очередь заменяется свежим списком инфлюенсеров с Notion-страницы.
- Добавлена ручная кнопка `Обновить` для загрузки Notion-списка без запуска скачивания.
- Парсер Notion поддерживает Instagram-ссылки, `@username`, подчёркивания и rich text форматирование Notion.
