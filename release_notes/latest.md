## SaveStories 0.4.25

### Изменения с 0.4.20 по 0.4.25

#### 0.4.20

- Добавлен выбор, что сохранять: только видео или фото и видео.
- Добавлено отображение прогресса в реальном времени по файлам и папкам.
- Улучшена пакетная выгрузка и общая стабильность worker runtime.

#### 0.4.21

- Новый экран `Главная` стал основным сценарием для пакетной выгрузки.
- Добавлены недавние наборы профилей для повторного запуска.
- Подготовлен отдельный раздел `Reels` как следующий этап развития интерфейса.

#### 0.4.22

- Улучшены release-сборки macOS: исправлена упаковка `Sparkle.framework` и fallback на встроенный Python runtime.
- Исправлено сохранение пользовательской папки выгрузки между перезапусками.
- Исправлены статусы пакетной очереди и корректный подсчёт обработанных профилей.
- Для Windows сохранена совместимость настроек после переименования приложения.

#### 0.4.25

- Полностью переработан интерфейс `Главной` на macOS: блоки `Status`, `Result`, `Fast Start`, `Недавнее`, `Очередь` и `Логи` собраны в новый рабочий layout.
- Улучшен узкий layout окна: экран дольше остаётся двухколоночным и аккуратнее перестраивается на маленькой ширине.
- Блок `Недавнее` показывает последний набор и умеет раскрывать остальные через кнопку `ещё`.
- В `Status` добавлены более живые индикаторы: светящаяся зелёная точка, анимация текущего шага и live-обновление шага по ходу пакетной выгрузки.
- Обновлён звук запуска кнопки `Скачать`, а подписи и порядок действий в `Fast Start` стали компактнее и понятнее.
- Версия синхронизирована для macOS и Windows, чтобы обе платформы можно было выпустить как единый релиз `0.4.25`.
