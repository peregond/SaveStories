## SaveMe 0.6.17

- В macOS добавлен отдельный раздел `Сортировка` с пошаговым сценарием: выбрать папку `Перенос`, папку назначения, при необходимости правила и затем разложить файлы
- После раскладки приложение может собрать готовый список файлов по папкам и скопировать его в буфер обмена
- Для файлов внутри Google Drive добавлено копирование ссылок: сначала через метаданные `drivefs`, а если их нет, через запасной путь через Finder
