## SaveMe 0.6.9

- Релиз включает macOS-версию и Windows WinUI 3 beta
- Для Windows WinUI 3 на GitHub публикуется установщик .exe с bundled Node runtime
- В настройках WinUI добавлен выбор темы: «Системная / Светлая / Тёмная» (по умолчанию используется системная тема)
- Улучшена чёткость интерфейса на масштабе Windows 125–150% (PerMonitorV2 DPI awareness)
- Исправлены ресайз окна и прокрутка контента на экранах Stories, Queue, Reels и Settings
- Обновлён визуальный стиль WinUI: добавлен мягкий лаймовый градиентный фон с полупрозрачными карточками
