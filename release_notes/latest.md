## SaveMe 0.6.44

- macOS Sorting: добавлена опция `Обновлять правила из Notion`.
- Windows Sorting: добавлена такая же опция и ручная кнопка `Обновить`.
- Перед сортировкой правила заменяются свежим списком с Notion-страницы, если тумблер включён.
- Правила с алиасами через `:` разворачиваются в отдельные строки; комментарии в скобках и `*...*` очищаются.
